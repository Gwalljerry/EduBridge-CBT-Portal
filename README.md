# EduBridge: The Digital Inclusion CBT Portal
**Submission for NextGen Knowledge Showcase (March 2026)**

### 🌎 Pillar: Digital Inclusion & Education
**Problem:** In my community, students fail national computer-based exams not because of a lack of intelligence, but because they have never interacted with a digital testing interface due to high data costs and poor connectivity.

### 💡 The Solution
**EduBridge** is a lightweight, mobile-responsive CBT portal that focuses on:
1. **Low Data Footprint:** Uses minimal assets to ensure loading on 2G/EDGE networks.
2. **Offline-Ready:** Built to function as a standalone tool that can be deployed on local Intranets.
3. **Accessibility:** Simplified UI designed for first-time computer users.

### 🛠 The Build
- **Frontend:** HTML5, Tailwind CSS (for modern, responsive design).
- **Logic:** Vanilla JavaScript for speed and compatibility.
- **Impact:** Automates grading and familiarizes students with digital environments.

### 🚀 Scalability
This project can be scaled to support 1,000+ students per school by integrating a local Raspberry Pi server, allowing entire villages to take mock exams without a single byte of global internet data.
